# Ushop App

A complete e-commerce mobile application built using Flutter and Firebase. It allows users to browse products, manage their carts, place orders, and more â€” all in a sleek and responsive interface.

---

## Features

- User authentication (Sign up / Login)
- Product browsing and filtering
- Add to cart and manage orders
- Wishlist functionality
- Admin product management
- Payment integration
- Order tracking
- Dark & Light theme support
- Responsive UI
- Real-time data with Firebase

---

## Tech Stack

- **Flutter**
- **Firebase (Auth, Firestore, Storage)**
- **GetX** for state management and routing
